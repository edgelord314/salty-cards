# Salty Cards
Salty Cards is a library for working with (currently) cards of 1 kind:
1. The French (Anglo-American) deck
    1. Hearts - ♥
    2. Tiles (Diamonds) - ♦
    3. Clovers (Clubs) - ♣
    4. Pikes (Spades) - ♠
    
## Features
Salty Cards provides several features to work with cards, such as the following:

- Create cards
- Store cards
